# FD-rate-calculator
The project is a web-based FD (Fixed Deposit) rate calculator that allows users to calculate the maturity amount based on the principal amount, selected bank, and time period. It provides a user-friendly interface for entering the required details and displays the calculated result.
Features:
Responsive design: The calculator is designed to be responsive, adapting to different screen sizes and providing an optimal user experience on various devices.
Bank selection: Users can choose from a list of banks to calculate FD rates specific to each bank.
Calculation: The calculator performs interest rate calculations based on the entered principal amount, selected bank's interest rate, and time period.
Technologies Used: HTML, CSS, JavaScript, VS code
